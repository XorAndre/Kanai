Ponto de partida

Este Framework e UI Kit foi criado para aqueles que estão no nível iniciante de Front-end e ainda desconhecem o poder de frameworks que tem um pré processador como Sass, Less e Stylus, nele você pode encontrar Grids, formulários, controle de imagem responsive, menu e alguns botões. Para usar o UI Kit baixe os arquivos CSS e JavaScript. Logo abaixo você tem duas opções com e sem pré processador, para iniciantes é comendado usar o arquivo com puro CSS, mas se você já é conhece de pré processadores então fique a vontade para baixar a versão de arquivo com Stylus.
